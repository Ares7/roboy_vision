Detailed Description

Intrinsic parameters of a camera.

Note
Similar to the CalibrationParameters, those parameters are taken from the settings file (SNXXX.conf) and are modified during the sl::Camera::open call (with or without Self-Calibration).
Those parameters given after sl::Camera::open call, represent the "new camera matrix" that fits/defines each image taken after rectification ( through retrieveImage).
fx,fy,cx,cy must be the same for Left and Right Camera once sl::Camera::open has been called. Since distortion is corrected during rectification, distortion should not be considered after sl::Camera::open call.
