Detailed Description

The class Matrix3f represent a generic three-dimensional matrix.



sl::Matrix3f is defined in a row-major order:
- It means that, in the value buffer, the entire first row is stored first, followed by the entire second row, and so on.
- you can access the data with the 'r' ptr or by element name as : r00, r01, r02 | 0 1 2 | r10, r11, r12 <-> r| 3 4 5 | r20, r21, r21 | 6 7 8 |
