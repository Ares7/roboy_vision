◆ depth_minimum_distance

float depth_minimum_distance

Specify the minimum depth information that will be computed, in the sl::UNIT you previously define.
default : 70cm (-1)

Warning
The computation time is affected by the value, exponentially. The closer it gets the longer the disparity search will take. In case of limited computation power, consider increasing the value.
Note
This value is used to calculate the disparity range estimation ( in pixels). Due to metric to pixel conversion, a small difference may occur with the value returned by getDepthMinRange().
